## 새로 알게된 것들  

- Model - View - Urls의 관계  
  - model이 ORM으로 편리하게 데이터를 처리한점이 다른 프레임워크들과는 다르게 너무 편리해서 좋았음.(덕분에 쉽게 실습을 진행할 수 있었던 것 같기도 함)
  - views.py를 통해서 내가 원하는 함수들을 편리하게 만드는 점 또한 신세계였다.
  - 이렇게 만든 view를 통해서 urls에 내가 원하는 경로를 등록만 하면 된다는 점 또한 너무 편했음
